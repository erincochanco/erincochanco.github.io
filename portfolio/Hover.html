Exercise

In order to familiarize ourselves with the hover function we were tasked to utilize it in a simple manner. One of my favorite memes on the internet are cat memes so I wanted to reference one I really like: angy cat. I'm fond of angy cat because it looks so sulky. Personally, I don't like being physically bothered when I'm such a mood. Due to this, I attached the text "Don't touch me!" because it's cohesive with the cat's seemingly sulky mood and added context to the illustration.

The hover function comes into play when you mouse over angy cat. It disappears as if it ran away to avoid contact.
